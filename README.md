# formcutter

ai-assisted i-864 affidavit of support form filler.

photo your docs (license, green card, tax transcript) → chat to fill gaps → get a filled pdf.
accredited-rep review in the loop.

## stack

next.js 15 · typescript · tailwind · shadcn/ui · claude sonnet 4.6 (vision + reasoning) · pdf-lib · prisma + sqlite

## dev

```bash
cp .env.example .env.local  # add your anthropic key
npm install
npm run dev
```
